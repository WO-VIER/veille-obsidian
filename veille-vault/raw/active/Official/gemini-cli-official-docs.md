# Gemini CLI official docs

## Documentation overview
Source : https://google-gemini.github.io/gemini-cli/docs/

Extrait capture :

> Gemini CLI apporte les capacites des modeles Gemini dans un environnement terminal de type REPL.

Points visibles dans la documentation :

- le projet se presente comme un agent terminal-first ;
- la doc met en avant l'interaction avec le systeme de fichiers, le shell et le web fetching ;
- la documentation cite aussi le mode headless, le checkpointing et les integrations avancees.

## Official GitHub README
Source : https://github.com/google-gemini/gemini-cli

Points visibles dans le README :

- projet open source sous licence `Apache-2.0` ;
- mise en avant d'un acces gratuit avec compte Google personnel ;
- contexte long, outils integres, support `MCP`, terminal-first et personnalisation via `GEMINI.md` ;
- cadence de release explicite avec canaux `preview`, `stable` et `nightly`.

## Authentication
Source : https://google-gemini.github.io/gemini-cli/docs/get-started/authentication.html

Points visibles dans la documentation :

- trois voies d'authentification principales : login Google, cle API Gemini, ou Vertex AI ;
- le login Google est presente comme la voie la plus simple pour un usage local ;
- la documentation distingue aussi le mode headless et les contextes entreprise.
