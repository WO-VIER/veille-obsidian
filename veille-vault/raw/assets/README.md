# Raw Assets

Ce dossier accueille les images, captures et pièces jointes téléchargées depuis les sources.

Règles :

- conserver les fichiers au format brut ;
- ne pas déplacer les assets référencés par des notes existantes ;
- utiliser ce dossier comme emplacement stable pour Obsidian Web Clipper et les téléchargements d'images.
