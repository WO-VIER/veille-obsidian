# Rituels de veille

Ce document décrit le rythme de fonctionnement du système.

## Quotidien

- Durée : 10 à 15 minutes.
- Actions :
  - parcourir les nouvelles entrées passives ou les nouveaux signaux entrants ;
  - supprimer mentalement le bruit sans culpabilité ;
  - déposer 1 à 3 sources prometteuses dans `raw/active/`.

## Hebdomadaire

- Durée : 45 minutes.
- Actions :
  - traiter les meilleures sources de `raw/active/` ou `raw/passive/` ;
  - produire au moins une vraie note dans `wiki/notes/` ;
  - relier cette note aux autres si des connexions existent ;
  - mettre à jour le radar et le journal de biais si nécessaire.

## Mensuel

- Durée : 30 minutes.
- Actions :
  - lancer un lint du wiki ;
  - repérer les pages orphelines ou obsolètes ;
  - nettoyer les doublons ;
  - réévaluer les sujets en `Surveiller` et `Expérimenter`.
