# Notes

Ce dossier contient toutes les vraies notes Obsidian du système de veille.

Règles simples :

- une note = un sujet clair ;
- une note peut être atomique ou synthétique ;
- les notes doivent être reliées quand des connexions existent ;
- les sources doivent rester visibles ;
- la structure doit rester simple et soutenable.
